ARCHI — скріншоти для Google Play
===================================

Рекомендації Google (телефон, 2024–2025):
  • Формат: PNG або JPEG, 16:9 або 9:16 (портрет зручний для стрічки).
  • Довжина сторони: від 320 px до 3840 px (найчастіше 1080×1920 або 1440×2560).
  • Мінімум 2 скріншоти; до 8 на тип пристрою.

Що зняти (гарна історія для користувача):
  1) Головна / привітання — видно назву ARCHI та навігацію.
  2) Новини — стрічка з постом.
  3) Оголошення (продаж/купівля) — картка оголошення.
  4) Чат — кілька повідомлень.
  5) Налаштування / мова — показує багатомовність (за бажанням).

Як зняти якісно:
  • Увімкніть мову інтерфейсу EN (або UK) у налаштуваннях перед знімком — однакова серія для стору.
  • Android: емулятор або пристрій → кнопки живлення+гучності, або Studio / scrcpy.
  • Якщо застосунок = WebView: знімайте сайт у Chrome DevTools (режим телефону) з однаковим розміром вікна для всіх кадрів.
  • Приберіть особисті дані (телефони в оголошеннях — тестові або заблюрьте).

Після знімків:
  • Перейменуйте файли: 01-home.png, 02-news.png, …
  • Завантажте в Play Console → Store listing → Phone screenshots.
